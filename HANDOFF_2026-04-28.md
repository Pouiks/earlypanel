## Pièges à éviter

- Ne PAS utiliser `taskkill /PID xxx /F` via Bash (MSYS path-mangling casse le flag) → utiliser PowerShell `Stop-Process -Id xxx -Force`
- Ne PAS oublier que dev local + DB prod = chaque seed crée vraiment des données prod (taggées `[E2E TEST]` / `@e2e.earlypanel.test`, mais visibles dans la table `staff_audit_log` à jamais — append-only)
- Pour générer un magic link admin sans email : `POST https://<ref>.supabase.co/auth/v1/admin/generate_link` avec service_role → récupérer `hashed_token` → naviguer sur `/app/auth/callback?token_hash=<hash>&type=magiclink` (le callback fait le `verifyOtp` côté SSR et set les cookies)
